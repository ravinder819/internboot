Objective:
To deploy the trained machine learning model as a web application using Flask, allowing users to input parameters and get sales predictions.

Features:
1.Dropdown-based inputs for error-free user interaction.
2.Backend validation for invalid dates.
3.Clean UI with HTML and CSS.
4.Model inference using trained regression model.

Technologies Used:
Flask, Python, Pandas, Scikit-learn, HTML & CSS

Outcome:
Successfully converted an ML model into a working web application, demonstrating end-to-end ownership from data to deployment.
